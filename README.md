# 핵 막 써

**핵 막 써**는 모바일 플레이를 우선으로 만든 HTML5 Canvas 기반 탑다운 액션 프로토타입입니다.  
플레이어는 적과 탄환을 피하면서 자동 공격으로 적을 처치하고, 위기 상황에서는 여러 가지 "핵" 능력을 사용해 전투를 뒤집을 수 있습니다.

> 핵을 쓰면 강해지지만, 탐지 게이지가 올라가며 점점 위험해지는 게임입니다.

## 현재 구현된 기능

- 모바일 터치 이동
- 자동 조준 / 자동 공격
- 적 추적 AI
- 적 탄환
- 플레이어 체력
- 피격 후 짧은 무적 시간
- Wave 진행 시스템
- 탐지 게이지
- 핵 버튼 3종
  - `BULLET DELETE`: 적 탄환 전부 삭제
  - `TIME STOP`: 일정 시간 적과 탄환 정지
  - `DAMAGE OVERFLOW`: 다음 공격 피해량 대폭 증가
- 탐지 60% 이상 / 100% 이상일 때 화면 경고 효과

## 조작 방법

모바일 기준:

- 화면 터치 / 드래그: 플레이어 이동
- 오른쪽 버튼: 핵 사용
- 공격: 가까운 적을 자동으로 조준해 자동 발사

## 실행 방법

별도의 빌드 과정 없이 실행할 수 있습니다.

1. 저장소를 다운로드하거나 파일을 클론합니다.
2. `index.html`을 브라우저에서 엽니다.
3. 모바일에서는 GitHub Pages 또는 브라우저 미리보기로 실행할 수 있습니다.

GitHub Pages 배포 시 예상 주소:

